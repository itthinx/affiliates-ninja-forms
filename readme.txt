== Description ==

This plugin integrates [Affiliates](http://wordpress.org/plugins/affiliates/) with [NinjaForms](https://wordpress.org/plugins/ninja-forms/).

With this integration plugin, referrals are created automatically for your affiliates when forms are sent.

*Affiliates* is an outstanding platform that helps to boost sales through your affiliate program and is the ideal affiliate solution for *WooCommerce*.

Requirements:

- [Affiliates](http://wordpress.org/extend/plugins/affiliates)
- [Affiliates Ninja Forms Integration](http://wordpress.org/extend/plugins/affiliates-ninja-forms) (this plugin)
- [NinjaForms](https://wordpress.org/plugins/ninja-forms/)

Install these, set up your shop, decide how much you want to pay your affiliates and start get money!

== Installation ==

1. Install and activate the [NinjaForms](https://wordpress.org/plugins/ninja-forms/) plugin.
2. Install and activate the [Affiliates](http://wordpress.org/extend/plugins/affiliates) plugin. Use the default settings or configure it to your needs.
3. Install and activate the [Affiliates Ninja Forms Integration](http://wordpress.org/extend/plugins/affiliates-ninja-forms) plugin.
4. A new *Ninja Forms Integration* menu item will appear under the *Affiliates* menu in WordPress. Set the amount and currency for your affiliates there.

Note that you can install the plugins from your WordPress installation directly: use the *Add new* option found in the *Plugins* menu.
You can also upload and extract them in your site's `/wp-content/plugins/` directory or use the *Upload* option.
